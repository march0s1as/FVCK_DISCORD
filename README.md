** pip install -r requirements.txt **

Como inserir o token?

   02. Inspecione elemento;
   03. Vá para Network;
   04. Atualize a página;
   05. Pesquise por "authorization";
   06. Copie seu token;
   07. use: python3 FVCKDISCORD.py <o token que voce copiou> 
   08. Usufrua do script.
